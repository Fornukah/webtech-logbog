Hjem Rysensteen Projekter Kontakt

Indtroduction

Vi skulle prøve at udvikle et spil, som vha MIT studerendes Makey-Makey, skulle få bevægelse tilbage i børns legeliv.
Inspirationen kunne komme fra spil som Fægtning, Dance, Dance Revolution.
Derefter skulle vii brainstorme, udvikle, code i Javascript og fremlægge et spil.



Teoretiske Overvejelser

Vi skulle overveje, blandt andet, hvordan vi bedst skulle kommunikere vores spil, og skabe den bedst mulige UX.
dette inkluderer f.eks, farvevalg, opsætning af indhold ift hvor man ligger scoreboardet m.m.
Vi skulle også overveje hvordan vi afbalncerede sværhedsgraden, i et spil, som skulle spilles fysisk.
F.eks hvor meget spilleren skulle rykke op og ned pr tryk af en knap, skruede vi meget på, så det både var naturligt og gennemskueligt, men også svært nok for spillererne.

            function setup() {
                createCanvas(1536,864); // Defines the game area
                image(background,0,0); // Loads the background
                PlayerOneY = 0.5*864-25; // Defines the start position on the Y axis for Player One
                PlayerOneX = 300; // Defines the start position on the X axis for Player One
                PlayerTwoY = 0.5*864-25; // Defines the start position on the Y axis for Player Two
                PlayerTwoX = 1536-300; // Defines the start position on the X axis for Player Two
                BallY = 0.5*864; // Defines the start position for the Y axis for the ball
                BallX = 0.5*1536-10; // Defines the start position for the X axis for the ball
                ScoreOne = 0; // Sets the start score for Player One
                ScoreTwo = 0; // Sets the start score for Player Two
                xSpeed = 0; // Defines the initial speed on the X axis for the ball
                ySpeed = 0; // Defines the initial speed on the Y axis for the ball
                Start = 0; // Defines that the game is not started
              }
