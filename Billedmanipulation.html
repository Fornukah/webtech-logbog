Billedmanipulation

Hjem Rysensteen Projekter Kontakt

Indtroduction

Vi skulle kombinere filtre for at forstå hvordan man kan ændrer på pixelværdier til at ændre på billedets indtryk.
Vi tog inspiration fra de givne filtre forarbejdet af vores lærer Mads Peter.
Derefter skulle vi brainstorme, code i Javascript og fremlægge vores filtre.

Link til Spil (det anbefales at zoome ud med den nødvendige mængde)


Teoretiske Overvejelser

Vi skulle overveje, blandt andet, hvordan man vha matematik
dette inkluderer f.eks, farvevalg, opsætning af indhold ift hvor man ligger scoreboardet m.m.
Vi skulle også overveje hvordan vi afbalncerede sværhedsgraden, i et spil, som skulle spilles fysisk.
F.eks hvor meget spilleren skulle rykke op og ned pr tryk af en knap, skruede vi meget på, så det både var naturligt og gennemskueligt, men også svært nok for spillererne.

Teoretiske Overvejelser I Praksis

I det nedenstående kodeeksempel ser vi netop hvordan vi definerer hvad der sker idet spilleren trykker på diverse Knapper. Vi definerede at spilleren ville rykke 45 pixels pr tryk, hvilket vi følte var den perfekte balance, og gav spilleren så stor en mulighed for at ramme bolden uden "blindspots".
Netop disse blindspots var et problem vi havde, at mellemrummet imellem tryk skabte steder som spilleren ikke kunne ramme.
