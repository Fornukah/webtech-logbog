Indtroduction

Vi skulle kombinere filtre for at forstå hvordan man kan ændrer på pixelværdier til at ændre på billedets indtryk.
Vi tog inspiration fra de givne filtre forarbejdet af vores lærer Mads Peter.
Derefter skulle vi brainstorme, code i Javascript og fremlægge vores filtre.

Teoretiske Overvejelser

Vi skulle overveje, blandt andet, hvordan man vha matematik kan redigere pixelværdier, netop værdierne for rød, grøn, blå og gennemsigtighed, for at ændre et billeds generelle udtryk.
Vi kunne f.eks, forskyde grønne og røde værdier til at give en "3d effekt", eller lave et mere kontrastrigt billede ved at enten maksimere eller minimere hver pixelværdi.

Teoretiske Overvejelser I Praksis

I det nedenstående kodeeksempel ser vi netop hvordan koden tager en enkel pixelværdi og maksimere hver værdie for så at skabe dette "kontrastrige" billede.
Hvad vi dog endte med at gøre var at vi ikke maksimerede værdierne, men kun lige tæt på, for at give et pænere og mere spændende billede at beskue. 

          
            function shadowWeight(threshold){
              //Forøg farve
              for (let k=0;k<3;k+=1){
                  if (c[k] > threshold) {
                      c[k]=255;
                    } 
                     //Formindsk farve
                    if (c[k] < threshold) {
                      c[k]=255/8;
                    }      
              }
              }

I kodeeksempelet ovenover kan man se hvordan vi har en threshold, som vi tidligere har defineret til 200 +- vores musposition. Derefter så tager vi for hver værdi og enten maksimerer den eller minimere den til en ottendedel af maksimum, netop 255.
I den 3 linje gør vi at den kører hver vores pixelværdi igennem, der er netop 3 værdier mellem 0-2. Så når vi siger at den skal køre + 1 hver gang, gør vi at den kører igennem hver pixelværdi, en efter en.
